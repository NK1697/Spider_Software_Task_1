# The King of Diamonds

This is a simple web app for the Spider R&D Software Task 1 basic app development task.

## How to run

Open `index.html` in a browser.

## What it does

- The user enters a whole number from 0 to 100.
- The bot randomly chooses a number from 0 to 100.
- The app calculates the average of both numbers.
- The Spider Number is calculated as `average * 0.8`.
- The player closest to the Spider Number wins the round.
